Rectangle image

A rectangular shape in an HTML image map is used to define a clickable area on an image. It is specified using the shape="rect" attribute in the tag. The coords attribute determines the boundaries of the rectangle with four numeric values: x1, y1, x2, and y2. These represent the x and y coordinates of the top-left corner (x1, y1) and the bottom-right corner (x2, y2) of the rectangle, measured in pixels relative to the top-left corner of the image. This allows users to interact with a specific rectangular region of an image, such as a button or a highlighted section, by clicking on it. When the user clicks within the defined rectangle, the associated link (href) or action is triggered.
